# Create App
Easily create Reboost enabled apps.

## Usage
```shell
npm init @reboost/app
```

## Available templates
- Electron
- Lit
- Malina.js
- Preact
- React
- React TypeScript
- Solid
- Svelte
- Vanilla
- Vue
